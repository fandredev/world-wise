## Clone this repo

###### Notes: Please, check the .tool-versions before use this project to show versions that i used.

```
git clone git@github.com:fandredev/world-wise.git
```

## Go to directory

```
 cd world-wise
```

## Install dependencies

```
 yarn
```

## Run server

```
 yarn run dev
```
